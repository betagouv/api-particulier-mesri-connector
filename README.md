API Particulier data connector for the MESRI data provider.

# Installation

- `cp .env.dist .env`
- adjust the `.env` with some dev values
- `pipenv install`

# Local development

- `pipenv run flask run`

# Deployment

The app is deployed on Etalab's [Dokku](https://github.com/dokku/dokku) instance.
